Awesome Ghost themes

23 Jul'14 · 314 words · 2 minute read
ghost-blog
themes · ghost-blog

I was looking for a nice and elegant theme for my newly created Ghost Blog. Finally, I selected few of them. I am sharing here the full list and corresponding github links. All these themes are free and open source. The github repositories have full description about how to install and configure the respective themes.

I felt, all these themes are simple but very intiutive.

  1. Crisp-ghost-theme by kathyqian : This theme looks super simple and crisp as name suggest. I am using it for this blog itself. I switched to Hugo recently. This theme exists for hugo as well but I have found something better.

https://github.com/kathyqian/crisp-ghost-theme

  • Clone of Crisp-ghost-theme by hbpasti by This is just a clone of above theme with few configuration changes. The only reason I listed out it separately is that the above theme didn’t work for me. I was getting error saying 500 Index.js not found. But this clone works absolutely fine as you can see :).

https://bitbucket.org/hbpasti/my-crisp-ghost-theme-config.git

  • Readium by svenread: Readium is also a very good theme and quite medium like. Simple and sweet. svenread has few other very good ghost themes but those are not free. checkout his website.

https://github.com/starburst1977/readium

  • StayPuft by davidlecina: I wanted to use theme but only thing I didn’t like was all the posts were showing in full content form. Although it can be configurable by editing templates in code but who cares when other options are available.

https://github.com/dlecina/StayPuft

  • Velox by dreyacosta: By far this particular theme is best suited for coding blogs. It’s very content focus and distraction free. It was first choice but it didn’t work for me because of some configuration issues.

https://github.com/dreyacosta/velox


Few more good themes I found out later:

  • Purity Demo and description I will test out this theme in few days.

  • Adamcooke Demo Source

Try out these themes and if you have your own collection do share them.
